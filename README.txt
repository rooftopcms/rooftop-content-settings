rooftop-content-settings adds default content to Rooftop CMS

== Description ==

rooftop-content-settings amends the default content for a multisite site in Rooftop CMS.

Track progress, raise issues and contribute at http://github.com/rooftopcms/rooftop-content-settings

== Installation ==

rooftop-content-settings is a Composer plugin, so you can include it in your Composer.json.

Otherwise you can install manually:

1. Upload the `rooftop-content-settings` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There is no step 3 :-)

== Frequently Asked Questions ==

= Can this be used without Rooftop CMS? =

Yes, it's a Wordpress plugin you're welcome to use outside the context of Rooftop CMS. We haven't tested it, though.
